ESPHome code for a 4 zone sprinkler system.

Hardware: esp based 4 relay controller, old rainbird water controller
Modification:
1) turn off power to existing water controller at circuit breaker
2) remove electronics from case, leave 120vac to 24vac transformer
3) connect power and circuits to esp based relay controller
4) upload this yaml via ESPHome
5) Program start and run times in Home Assistant
